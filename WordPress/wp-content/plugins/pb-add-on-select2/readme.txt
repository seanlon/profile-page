== Description ==

Profile Builder - Select2 Add-On

Add an improved select field to Profile Builder fields list.

This add-on allows you to create select fields with search and filter functionality. All of this in a good looking, responsive select box.


NOTE:

This plugin requires and is compatible with Profile Builder version 2.0 or higher

== Installation ==

1. Upload and install the zip file via the built in WordPress plugin installer.
2. Activate the plugin from the "Plugins" admin panel using the "Activate" link.
3. Go to Manage Fields tab under Profile Builder menu and you'll have access to Select2 field.
