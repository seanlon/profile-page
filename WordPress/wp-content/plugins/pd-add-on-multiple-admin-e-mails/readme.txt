== Description ==

Profile Builder - Multiple Admin E-mails

Extends the functionality of Profile Builder by allowing you to set multiple admin e-mail addresses that will receive e-mail notifications sent by Profile Builder

NOTE:

This plugin is compatible with Profile Builder version 2.0.6 or higher
	


== Installation ==

1. Upload the profile-builder-multiple-admin-emails folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
