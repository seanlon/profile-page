== Description ==

Profile Builder - Email Confirmation Field

This add-on adds an Email Confirmation field to the Manage Fields list from Profile Builder.

NOTE:

This plugin is compatible with Profile Builder Hobbyist or Pro, version 2.0.9 or higher.



== Installation ==

1. Upload the profile-builder-email-confirmation-field folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
