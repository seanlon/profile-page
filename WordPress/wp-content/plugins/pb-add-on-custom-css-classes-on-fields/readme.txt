== Description ==

Profile Builder - CSS Classes for Fields

Extends the functionality of Profile Builder by allowing you add custom css classes for fields.

NOTE:

This plugin is compatible with Profile Builder version 2.1.3 or higher
	


== Installation ==

1. Upload the pb-add-on-custom-css-classes-on-fields folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
